YouTube Subtitle Generator

Python Flask OpenAI Whisper yt-dlp FFmpeg AI/ML Speech Recognition

Project Overview

YouTube Subtitle Generator is a web application that automatically generates and translates subtitles for YouTube videos using OpenAI's Whisper AI model. The application provides a synchronized viewing experience with real-time subtitle highlighting and auto-scrolling capabilities.

Built with Flask and modern web technologies, the application supports both regular YouTube videos and YouTube Shorts, automatically translating content from any language to English. It offers an accessible solution for content creators, educators, and viewers who need accurate subtitles for video content.

Core Features

  • AI-Powered Transcription

    Utilizes OpenAI's Whisper model for highly accurate speech recognition and automatic transcription of video content.

  • Automatic Translation

    Translates subtitles from any language to English automatically, making content accessible to global audiences.

  • Synchronized Playback

    Real-time subtitle synchronization with video playback using YouTube's IFrame API for seamless viewing experience.

  • Auto-scrolling Subtitles

    Intelligent subtitle highlighting and auto-scrolling that follows video playback for enhanced readability.

  • YouTube Shorts Support

    Full compatibility with both regular YouTube videos and YouTube Shorts format for comprehensive coverage.

  • Responsive Design

    Mobile-friendly interface that adapts to various screen sizes for optimal viewing on any device.

Technical Architecture

AI & Machine Learning Components

  • OpenAI Whisper: State-of-the-art speech recognition model for accurate transcription
  • Automatic Language Detection: Identifies source language and translates to English
  • Timestamp Generation: Precise timing information for subtitle synchronization
  • Audio Processing: Advanced audio extraction and preprocessing for optimal recognition

Backend Infrastructure

  • Flask Web Server: Lightweight Python framework handling requests and processing
  • yt-dlp Integration: Robust YouTube video downloading and audio extraction
  • FFmpeg Processing: High-quality audio format conversion and optimization
  • File Management: Automatic cleanup of temporary files after processing

Frontend Features

  • YouTube IFrame API: Embedded video player with full playback controls
  • Real-time Synchronization: JavaScript-based subtitle timing and highlighting
  • Progressive Enhancement: Graceful degradation for different browser capabilities
  • Responsive Layout: CSS Grid and Flexbox for optimal display across devices

Processing Workflow

Audio Extraction Pipeline

  • URL Validation: Verifies YouTube URL format and video availability
  • Audio Download: Extracts high-quality audio stream using yt-dlp
  • Format Conversion: Converts audio to optimal format for Whisper processing
  • Quality Optimization: Preprocesses audio for improved recognition accuracy

AI Processing Steps

  • Speech Recognition: Whisper model processes audio for text transcription
  • Language Detection: Automatic identification of spoken language
  • Translation Processing: Converts non-English content to English text
  • Timestamp Alignment: Generates precise timing for subtitle synchronization

Output Generation

  • Subtitle Formatting: Creates timestamped subtitle entries
  • Web Integration: Embeds processed subtitles with YouTube player
  • Cleanup Process: Removes temporary files and frees system resources

System Requirements

Core Dependencies

  • Python 3.11: Specifically required for OpenAI Whisper compatibility
  • FFmpeg: Essential for audio processing and format conversion
  • CUDA Support: Optional GPU acceleration for faster processing
  • Internet Connection: Required for YouTube video access and downloads

Performance Considerations

  • Processing Time: Varies based on video length and system capabilities
  • Memory Usage: Proportional to video duration and audio quality
  • Storage Requirements: Temporary space for audio files during processing
  • Network Bandwidth: Affects download speed and processing initiation

Installation & Setup

Platform-Specific Setup

  • Ubuntu/Debian: sudo apt-get install ffmpeg
  • Arch Linux: sudo pacman -S ffmpeg
  • macOS: brew install ffmpeg
  • Windows: Download from FFmpeg official website

Virtual Environment Setup

  • Create environment specifically with Python 3.11
  • Install dependencies from requirements.txt
  • Verify FFmpeg installation and PATH configuration
  • Test Whisper model download and initialization

Development Challenges & Solutions

Python Version Compatibility

Challenge: OpenAI Whisper requires specific Python versions, with newer versions causing dependency conflicts.

Solution: Specified Python 3.11 requirement and provided clear installation instructions for virtual environment setup with version constraints.

Real-time Synchronization

Challenge: Achieving precise synchronization between YouTube video playback and generated subtitle timing.

Solution: Implemented JavaScript timing mechanisms with YouTube IFrame API integration for frame-accurate subtitle display.

Audio Processing Performance

Challenge: Optimizing processing time for long videos while maintaining transcription accuracy.

Solution: Utilized FFmpeg optimization settings and Whisper model selection based on accuracy vs. speed requirements.

Cross-Platform Compatibility

Challenge: Ensuring consistent functionality across different operating systems and FFmpeg installations.

Solution: Provided comprehensive installation guides for each platform and implemented robust error handling for missing dependencies.
