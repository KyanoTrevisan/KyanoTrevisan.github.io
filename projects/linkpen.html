Linkpen

Node.js Express EJS JavaScript CSS3 Authentication Web App

Project Overview

Linkpen is a free, open-source, customizable link-in-bio tool that helps creators share everything they create, curate, or sell from their social media profiles. Built with privacy and user control in mind, it offers a self-hosted alternative to commercial link-in-bio services.

The platform features email-less account creation, highly customizable themes, and a comprehensive admin dashboard for managing users and content. Originally titled "friendly system," Linkpen has evolved into a robust platform for content creators and businesses.

Core Features

  • Email-less Registration

    Create accounts without requiring email addresses, enhancing user privacy and reducing barriers to entry.

  • Role-Based Access Control

    Sophisticated user management with Admin and standard user roles, with the first user automatically becoming instance admin.

  • Custom Themes & Templates

    Highly customizable themes and templates that users can personalize to match their brand and style preferences.

  • Admin Dashboard

    Comprehensive admin interface for managing users, removing bad actors, and promoting/demoting user privileges.

  • Security Features

    Strict password requirements and built-in security measures to protect user accounts and data.

  • Custom Logging

    Advanced debugging capabilities with custom logging systems for monitoring and troubleshooting.

Development Challenges & Solutions

Email-less Authentication

Challenge: Creating a secure authentication system without requiring email addresses while maintaining account recovery capabilities.

Solution: Implemented username-based authentication with strong password requirements and admin-mediated account recovery options.

Theme System Architecture

Challenge: Building a flexible theme system that allows extensive customization while maintaining security and performance.

Solution: Developed a template-based system using EJS with JSON configuration files, allowing safe customization without compromising platform security.

User Management & Security

Challenge: Balancing user freedom with platform security and preventing abuse from bad actors.

Solution: Implemented comprehensive admin tools, role-based access control, and monitoring systems to quickly identify and address problematic users.
