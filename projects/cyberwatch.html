CyberWatch - Threat Intelligence

Python FastAPI Ollama Telethon SQLite BeautifulSoup

Project Overview

A comprehensive, open-source threat intelligence platform designed for monitoring cybercrime communities, forums, and marketplaces across both the clear and dark web. Built with privacy and security in mind, CyberWatch uses local AI models for threat analysis without sending data to external services, making it perfect for security research and SOC operations.

Real-time Telegram Monitoring

Monitors 200+ threat intelligence channels, including major threat actor groups, exploit databases, and security research communities.

AI-Powered Threat Analysis

Integrates Ollama with llama3.2:3b model for complete privacy-focused AI analysis. Automatically generates professional threat summaries, extracts IoCs (IPs, domains, hashes), and provides confidence scoring for threat prioritization.

Automated Threat Intelligence Feed

Automatically posts AI-generated threat summaries to Telegram channels, creating a professional threat intelligence feed. Features smart filtering, rate limiting, and confidence-based posting to prevent spam while ensuring high-quality intelligence sharing.

Technical Implementation

Backend Architecture

  • FastAPI framework with async/await for high-performance concurrent scanning
  • SQLite database with threat correlation and posting logs
  • Telethon for Telegram API integration and real-time monitoring
  • Ollama integration for local AI processing without external dependencies

AI & Machine Learning

  • Local llama3.2:3b model integration for threat classification
  • Automatic IoC extraction using regex patterns and AI analysis
  • Confidence scoring system combining keyword matching and AI assessment
  • Privacy-focused processing - no data sent to external services

Key Features

  • Real-time Monitoring

    24/7 automated scanning of threat intelligence sources with configurable scan intervals and keyword-based filtering.

  • Privacy-First AI

    Local Ollama AI processing ensures no sensitive threat data leaves your infrastructure while providing enterprise-grade analysis.

  • Automated Intelligence Feed

    AI-generated professional threat summaries automatically posted to your Telegram channel for team consumption.

Challenges & Solutions

Scale & Performance

Challenge: Efficiently monitoring 200+ sources without overwhelming APIs or system resources.

Solution: Implemented intelligent rate limiting, concurrent processing with asyncio, and smart filtering to process only relevant content.

AI Integration & Privacy

Challenge: Providing threat analysis while maintaining complete data privacy.

Solution: Integrated Ollama for local AI processing, ensuring no threat intelligence data is sent to external services.

Data Quality & Noise Reduction

Challenge: Filtering high-quality threats from massive amounts of social media noise and false positives.

Solution: Developed multi-layered filtering using keyword matching, AI confidence scoring, and threat level classification.
