OSINT Studio

Rust React TypeScript Tauri 2.0 Cytoscape.js Desktop App Graph Visualization OSINT

Project Overview

OSINT Studio is an open-source desktop application designed for Open Source Intelligence (OSINT) investigations. Built with Rust and React using the Tauri 2.0 framework, it provides intelligence analysts and researchers with a comprehensive tool for visualizing relationships, tracking entities, and generating investigation reports.

The project was created to address the lack of quality Free and Open Source Software (FOSS) alternatives in the OSINT space, providing a professional-grade tool that respects user privacy and data sovereignty.

Core Features

  • Interactive Graph Visualization

    Advanced drag-and-drop interface with multiple layout algorithms for relationship analysis.

  • Multi-Entity Support

    Track people, organizations, cryptocurrency wallets, social accounts, domains, and more.

  • Confidence-Based Relationships

    Color-coded arrows with percentage scores and source attribution for reliability tracking.

  • Advanced Search

    Full-text search across all entities and metadata for quick information retrieval.

  • Rich Metadata

    Attach descriptions, tags, confidence scores, and investigative notes to all entities.

  • Automated Reporting

    Generate professional investigation reports with multiple templates and formats.

Graph Layout Algorithms

OSINT Studio supports multiple graph layout algorithms to help visualize relationships in different ways, allowing analysts to discover patterns and connections more effectively.

Circle Layout

Arranges entities in a circular pattern, useful for showing equal relationships between all nodes.

Force-Directed Layout

Uses physics simulation to naturally cluster related entities and separate unrelated ones.

Grid Layout

Organizes entities in a structured grid pattern for systematic analysis and comparison.

Organic Layout

Creates natural, tree-like structures that emphasize hierarchical relationships.

Automated Report Generation

OSINT Studio includes powerful report generation capabilities that automatically compile investigation data into reports suitable for documentation. The reporting system generates multiple report types including investigation summaries, executive briefs, and detailed timeline analysis.

Technical Architecture

Backend (Rust)

  • Tauri 2.0: Cross-platform desktop framework providing secure API access
  • Serde: JSON serialization for efficient data exchange between frontend and backend
  • UUID: Unique entity identification ensuring data integrity
  • Chrono: Comprehensive date and time handling for timeline features
  • Anyhow: Robust error handling and recovery mechanisms

Frontend (React/TypeScript)

  • React 18: Modern UI framework with hooks for efficient state management
  • TypeScript: Type-safe development ensuring code reliability
  • Cytoscape.js: Interactive graph visualization with advanced layout algorithms
  • Lucide React: Modern icon library for consistent UI design
  • Vite: Fast build tool and development server

OSINT Capabilities

Entity Types Supported

  • Person: Individuals with biographical and contact information
  • Organization: Companies, agencies, and institutional entities
  • CryptoWallet: Cryptocurrency addresses and transaction tracking
  • SocialAccount: Social media profiles and online identities
  • Domain: Websites and internet infrastructure
  • IpAddress: Network infrastructure and hosting information
  • Email: Email addresses and communication patterns
  • Phone: Phone numbers and telecommunications data
  • Document: Files, reports, and evidence materials
  • Event: Incidents, meetings, and timeline markers

Relationship Types

  • Owns: Property, asset, or ownership relationships
  • Controls: Authority or control relationships
  • TransactsWith: Financial or transaction relationships
  • MemberOf: Membership in organizations or groups
  • ConnectedTo: General connections and associations
  • SameAs: Identity correlation across different platforms
  • RelatedTo: General relationships and connections
  • ParentOf/ChildOf: Hierarchical relationships

Security and Privacy

Local Data Storage

All investigation data is stored locally on the user's machine, ensuring complete control over sensitive information and eliminating cloud dependency risks.

Open Source Transparency

The entire codebase is open source under the MIT license, allowing security audits and community contributions to maintain high security standards.

Ethical OSINT Practices

The tool is designed for legitimate OSINT research using publicly available information only, with built-in reminders about legal and ethical guidelines.
